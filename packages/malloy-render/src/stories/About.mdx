# Malloy Renderer

The Malloy Renderer is a web component for visualizing the nested results of a Malloy view. Visualizations can be configured using [Malloy's annotation system](https://docs.malloydata.dev/documentation/language/tags). This application documents some of the possible outputs of the Malloy renderer. Use the left navigation panel to explore different types of Malloy outputs.
